# 用户使用手册

## 概述

曹二听说101是一款英语听说考试系统，可用于：
- **学生考试**：在电脑上进行英语听说测试，包含看题、听音频、录音作答
- **教师出卷**：使用模板快速创建标准化试卷
- **教师批改**：对学生的录音作答进行评分，导出成绩报告

## 界面导航

应用启动后，左侧边栏提供以下导航（`src/renderer/src/components/Layout.tsx`）：

| 页面 | 图标 | 功能 |
|------|------|------|
| 试卷列表 | ScrollText | 导入、查看、删除考试包，开始考试 |
| 作答列表 | Mic | 查看学生作答记录，导出作答包 |
| 创建试卷 | FilePlus | 基于模板创建试卷草稿 |
| 批改管理 | ClipboardCheck | 导入作答包、批改评分、查看批改记录 |
| 关于 | Info | 许可证信息、开发者模式开关 |
| 开发者选项 | Wrench | 数据管理（仅开发者模式可见） |

## 一、考试管理

### 导入考试

1. 点击"试卷列表"页面上的"导入考试"按钮
2. 选择一个 ZIP 格式的考试包
3. 系统自动验证格式并导入

考试包 ZIP 应包含 `exam.json` 和 `media/` 目录。格式要求见 [exam-format.md](exam-format.md)。

### 开始考试

1. 在试卷列表中，点击目标考试卡片的"开始考试"
2. 输入学生姓名和 6 位学号
3. 按题目顺序作答：
   - **准备题**（倒计时）：屏幕显示文本或图片，倒计时结束自动进入下一题
   - **录音题**：听到"准备录音"提示音后开始作答，录音自动限时
   - **播放题**：自动播放音频或视频，播放完毕自动进入下一题
4. 全部题目完成后显示"考试完成"
5. 录音文件自动保存，可在作答列表中查看和导出

### 导出考试

在试卷列表点击"导出"，可将已导入的考试重新打包为 ZIP。

## 二、试卷创建（模板 → 草稿 → 导出）

### 查看模板

点击"创建试卷"，左侧标签切换到"模板列表"，查看所有可用模板。

### 创建草稿

1. 选择一个模板，点击"创建草稿"
2. 系统基于模板创建草稿，自动跳转到草稿编辑页面

### 编辑草稿

草稿编辑页面显示模板中定义的可编辑内容：
- **文本字段**：直接输入文本（如朗读句子、情景描述等）
- **文件字段**：点击上传图片、音频等文件（支持剪贴板粘贴）
- **标题**：点击铅笔图标可自定义试卷标题

### 导出试卷

编辑完成后，点击"导出试卷"按钮：
1. 系统校验所有必填项是否完整
2. 对模板中的音频节点自动合成语音（TTS）
3. 生成标准格式的考试包 ZIP 文件
4. 导出后可继续编辑草稿

## 三、批改管理

### 导入作答包

1. 先在"作答列表"页面将学生的作答打包导出为 ZIP
2. 切换到"批改管理"页面，点击"导入作答包"
3. 选择作答 ZIP 文件
4. 系统自动去重，导入进度通过进度条显示

### 批改评分

1. 在"批改列表"标签中，勾选需要批改的作答记录
2. 点击"开始批改"
3. 进入批改会话页面：
   - **左侧**：题目信息（Markdown 渲染）和评分标准（可折叠）
   - **右侧**：音频播放器 + 评语文本框 + 分数输入
4. 输入分数和评语后点击"确认"，自动跳转到下一评分项
5. 可中途"暂停批阅"，已打分数会保存
6. 全部完成后点击"完成批阅"

### 结算

批改完成后进入结算页面：
- 显示每条记录的批改进度（已打分/总数）
- **本次结算**：创建批次，计算总分，完成批改
- **下次结算**：暂不结算，保留评分数据

### 导出成绩

在"批改记录"标签中，展开批次后可以：
- **导出表格**：生成 CSV 文件，包含每道题的分数和总分（UTF-8 with BOM，Excel 可直接打开）
- **导出 PDF**：为每个学生生成一份 PDF 成绩报告，打包为 ZIP

## 四、作答管理

在"作答列表"页面可以：
- 按姓名、学号、试卷名称搜索作答记录
- 勾选一条或多条记录后：
  - **导出**：打包为 ZIP 供批改使用
  - **删除**：移除作答记录
- 也可单独导出/删除某条记录

## 快捷键与提示

- **F12**：在开发者模式下打开 DevTools（需在"关于"页面开启开发者模式）
- 考试过程中，开发者模式会显示题目序号和"跳过"按钮
- 窗口大小变化时，考试界面自动等比缩放（3:2 比例，1200×800 设计尺寸）

## 数据存储位置

所有数据（考试、作答、模板、草稿、批改）存储在应用的用户数据目录下：

- **Windows**：`%APPDATA%/cyez-ls101/`
- **macOS**：`~/Library/Application Support/cyez-ls101/`
- **Linux**：`~/.config/cyez-ls101/`

可通过"开发者选项"页面打开此目录或重置所有数据。详见 [data-storage.md](data-storage.md)。
